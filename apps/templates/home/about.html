O que é ReqM4IoT?

ReqM4IoT é uma ferramenta online desenvolvida para apoiar desenvolvedores (principalmente aqueles que estejam desenvolvendo sistemas Internet of Things - IoT) na realização das etapas de elicitação e análise de requisitos e modelagem de software. Desenvolvedores distribuídos geograficamente podem trabalhar juntos em um espaço de trabalho compartilhado, organizando os requisitos e os diagramas.

Além de fornecer técnica especificas para analise de requisitos para IoT () e modelagem de software para IoT (SysIoTML), o sistema oferece várias funcionalidades que podem auxiliar e melhorar processo de desenvolvimento de software.


O que é SysIoTML?

O SysIoTML é uma extensão do SysML (Systems Modeling Language). A SysML é uma linguagem de propósito geral para modelagem de sistemas de aplicações de engenharia. O SysIoTML considerou aspectos específicos de Sistemas IoT (comportamento e interatividade), através dos diagramas: de Caso de Uso, Sequencia, Máquina de Estado e Bloco.[1]
